# daivalue

**daivalue** — сервис, который собирает информацию из разных источников и формирует из неё единый поток данных. Он подключается к внешним каналам (чаты, репозитории, документы), извлекает содержимое — тексты, файлы, изображения и другие материалы — и сохраняет его в исходном виде вместе с метаданными и привязкой к источнику. Сервис отслеживает прогресс обработки, знает, какие данные уже получены, и может повторно обходить источник для проверки и догрузки пропущенного.

Архитектура построена на системе плагинов: ядро отвечает за общий процесс сбора и хранения, а каждый источник подключается отдельным плагином, чтобы расширять список каналов без изменения ядра. На первом этапе реализуются универсальное ядро и плагин для чтения публичных Telegram-каналов.

На выходе daivalue формирует единый поток сырого материала с метаданными для последующих систем обработки и анализа.
