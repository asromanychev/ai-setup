# Flows And Templates Index

Каталог `memory-bank/flows/` содержит reusable process-layer: lifecycle rules, taxonomy стабильных идентификаторов и governed templates.

- [Task Workflows](workflows.md) — маршрутизация задач по типам и базовый цикл разработки.
- [Feature Flow](feature-flow.md) — lifecycle от draft до closure, gates и стабильные ID (`REQ-*`, `CHK-*`, `STEP-*` и др.).
- [Templates Index](templates/README.md) — эталонные шаблоны governed-документов: PRD, use case, ADR.

> Шаблоны для issue-артефактов (`brief`, `spec`, `plan`) живут в [`../templates/`](../templates/).
