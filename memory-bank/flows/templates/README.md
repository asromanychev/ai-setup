# Templates Index

Каталог `memory-bank/flows/templates/` хранит governed wrapper-шаблоны для документов уровня инициатив и решений.

- [`prd/PRD-XXX.md`](prd/PRD-XXX.md) — шаблон Product Requirements Document.
- [`use-case/UC-XXX.md`](use-case/UC-XXX.md) — шаблон канонического use case.
- [`adr/ADR-XXX.md`](adr/ADR-XXX.md) — шаблон Architecture Decision Record.

> Шаблоны для issue-артефактов (`brief`, `spec`, `plan`) живут в [`../../templates/`](../../templates/).
