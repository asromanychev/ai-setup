## Замечание 1 — Неопределённый термин в КУ #4

**Цитата:**
> «несколько контрольных точек, каждая из которых претендует на использование для следующего возобновления»

**Проблема:** Слово «претендует» вводит термин, не определённый в доменном контексте. Доменный контекст закрепляет инвариант: у источника существует либо 0, либо 1 контрольная точка — и эта единственная точка по определению используется для возобновления. Никакого разделения контрольных точек на «претендующие» и «не претендующие» в документе нет. Читатель вынужден догадываться о значении термина именно там, где описывается защитное поведение при нарушении инварианта.

**Предложение:** Убрать уточнение «каждая из которых претендует на использование для следующего возобновления», опираясь на уже определённый инвариант:

> «Если система сталкивается с состоянием, в котором для одного источника одновременно существуют несколько контрольных точек, она сигнализирует об ошибке и не продолжает возобновление по такому источнику как по однозначно определённому.»
